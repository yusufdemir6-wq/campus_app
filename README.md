# Campus App 🎓

A university mobile application built with Flutter and Dart that helps students manage their academic life in one place.

## About
Campus App allows students to log in with their student credentials, view their active courses for the semester, message professors and classmates, and check their academic profile including GPA, attendance rate, and course grades.

## Features
- Student login with student number and password
- Home screen displaying all active courses with teacher names
- Messaging screen with separate sections for professors and friends
- Profile screen showing GPA, attendance, credits, and current grades
- Settings menu inside the profile screen

## Screens
- `login_screen.dart` — student login page
- `home_screen.dart` — active courses list
- `messages_screen.dart` — chat with professors and friends
- `profile_screen.dart` — student profile, grades and settings
- `main.dart` — app entry point and navigation setup

## How to Run

### Requirements
- Flutter installed on your machine → [flutter.dev](https://flutter.dev)
- An Android device or emulator

### Steps
1. Clone this repository or download it as a ZIP
2. Open the project folder in VS Code
3. Open the terminal and install dependencies:
```
flutter pub get
```
4. Connect your Android device or start an emulator, then run:
```
flutter run
```
