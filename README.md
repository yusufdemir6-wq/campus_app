# Campus App
A university mobile app built with Flutter.

## Features
- Login screen with student number and password
- Home screen showing active courses
- Messaging system with professors and friends
- Profile screen with GPA, attendance and course grades and settings etc.

## Screens
- `login_screen.dart` — student login
- `home_screen.dart` — active courses list
- `messages_screen.dart` — chat with professors and friends
- `profile_screen.dart` — student profile and settings

## How to Run
1. Make sure Flutter is installed — flutter.dev
2. Clone or download this project
3. Open terminal in the project folder and run:
flutter pub get
flutter run
